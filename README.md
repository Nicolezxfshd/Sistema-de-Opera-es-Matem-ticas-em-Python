Como usar a calculadora

Ao executar o programa, o usuário deve inserir dois números que serão utilizados nas operações matemáticas. Após informar os valores, o sistema apresenta um menu com as operações disponíveis.

O usuário deve escolher a operação desejada digitando o número correspondente à opção exibida no menu. O programa então realiza o cálculo utilizando os números informados e exibe o resultado na tela.

As opções disponíveis são:

Soma

Subtração

Multiplicação

Divisão

Sair do programa

Após a execução de uma operação, o menu é exibido novamente para que o usuário possa realizar novos cálculos. Caso o usuário escolha a opção Sair, o programa será encerrado.

O sistema também possui tratamento de erros para entradas inválidas. Caso seja digitado um valor que não seja numérico ou seja realizada uma tentativa de divisão por zero, o programa exibirá uma mensagem informando o erro.
